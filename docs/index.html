OpenRBC

A lipid-resolution molecular dynamics simulator of human red blood cells

View on GitHub Download .zip Download .tar.gz

Intro

OpenRBC is an open-source molecular dynamics software for the human red blood cells. In fact, it is the first ever simulator that can simulate an entire red blood cell using coarse-grained molecular dynamics — using just a single workstation. This is achieved through the invention of a highly efficient adaptive spatial searching algorithm as well as the development of robust coarse-grained molecular dynamics force field. It makes efficient use of a large amount of CPU cores with hundreds of OpenMP threads.
